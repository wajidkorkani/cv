# CV - Curriculum Vitae Project

## Overview
A modern, interactive CV (Curriculum Vitae) built with HTML, CSS, and JavaScript. This project showcases a professional resume/CV presentation with a clean, responsive design.

## Project Structure
- **index.html** - Main HTML file containing the CV structure and content
- **index.css** - Stylesheet for styling and layout
- **index.js** - JavaScript file for interactive features and functionality

## Features
- Clean and professional design
- Responsive layout
- Interactive elements powered by JavaScript
- Easy to customize and update

## Getting Started
1. Clone or download the project
2. Open `index.html` in your web browser
3. Customize the content in `index.html` with your own information
4. Modify `index.css` to match your preferred styling
5. Add additional interactivity through `index.js`

## Usage
Simply open the `index.html` file in any modern web browser to view your CV. The page will display your professional information in a formatted layout.

## Customization
- Edit the HTML content in `index.html` to add your personal details
- Modify colors and fonts in `index.css`
- Add JavaScript interactions in `index.js` as needed
